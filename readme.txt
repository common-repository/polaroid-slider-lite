A nice looking polaroid-style image/video slider.

== Description ==

Polaroid slider Lite mimics the feeling of the old-style polaroid photographs. Its a powerful javascript-css slider using CSS3 and jQuery.

**Features List:**

* Video/Image/HTML slides
* Fully customisable slide label with HTML content
* Smooth slide animations with Ken Burns Effect
* Supports all major browsers (IE7+, FF, Chrome, Safari...) and mobile devices (Windows Phone, Android, IOS)
* Uses jQuery for maximum compatibility

== Installation ==

1. Upload `polaroid-slider-lite` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode from the settings into your template or post-page
